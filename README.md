Создайте приложение Django для управления задачами, где пользователи могут:

- Просматривать список задач.
- Создавать новую задачу.
- Удалять существующую задачу.

Требования:

Реализовать маршруты для следующих операций:
- /tasks/ – просмотр всех задач.
- /tasks/add/ – добавление новой задачи.(add_task)
- /tasks/delete/id/ – удаление задачи по идентификатору.

Опишите класс задачи:

- заголовок задачи
- текст задачи
- дата начала
- дата завершения
